#项目中使用的框架
* 项目开发使用的是spring-boot
* 持久层框架，使用的是 Spring Data
* NoSql框架使用的是 Redis
* 项目前台使用vue

#项目前后端关系
>>项目前端和后端分离，前端都是以vue编写，后端是用java编码，controller 都使用 @ResetController注解，前台访问之后返回json数据给前台。
